# **REGEX MASS EDITING**

---

> [!help]
> I am using Notepad++ I need a regex to fix a large number of punctuation issues with the period symbol. The file uses one space after a period. I'd like two period. Can you show me how?

I can help you with that! Here's a regex solution for Notepad++:

**Find what:** `\.(\s)(?=[A-Z])`

**Replace with:** `.` (that's period followed by two spaces)

This pattern:

- `\.` matches a literal period
- `(\s)` captures one whitespace character
- `(?=[A-Z])` looks ahead to ensure the next character is an uppercase letter (start of a new sentence)

**Steps in Notepad++:**

1. Press `Ctrl+H` to open Find & Replace
2. Paste the regex above into "Find what"
3. Type `.` (period + two spaces) in "Replace with"
4. Select "Regular expression" at the bottom of the dialog
5. Click "Replace All"

**Note:** If you want to catch ALL periods followed by single spaces (not just before capital letters), you can use the simpler pattern:

**Find what:** `\.`

**Replace with:** `.`

This will replace every period-single-space with period-double-space throughout the document. The first regex is safer because it only targets sentence endings, while the second will also change things like abbreviations (e.g., "U.S. policy" might become "U.S. policy").

Which approach works better depends on your specific content!

**Find what:** `\?`

**Replace with:** `?` (question mark followed by two spaces)

And yes, you need to escape the question mark with a backslash because `?` is a special character in regex (it means "zero or one of the preceding character").

If you want to be more selective and only match question marks at sentence endings (before capital letters), you can use:

**Find what:** `\?(\s)(?=[A-Z])`

**Replace with:** `?`

The same logic applies to exclamation marks too:

**Find what:** `!`

**Replace with:** `!` (exclamation mark followed by two spaces)

No escaping needed for `!` since it's not a special regex character.  You could run all three replacements in sequence to fix periods, question marks, and exclamation marks throughout your document.
